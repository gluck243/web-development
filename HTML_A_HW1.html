General Tags

To begin with, html file begins with <html> and </html>. They are used to create an html page. Then tags <head> and </head> set off the title & other info that isn't displayed.
<body> and </body> are used to create a visible part of the document.

Formatting

Tags <h> and </h> create a heading. Number (from 1 to 6) define the size of the heading.
<p> and </p> create paragraphs of the document. Also, tag <br> helps
